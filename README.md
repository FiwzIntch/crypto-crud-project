## Description

Crypto CRUD Project

### Install Postgres By Docker-compose
```bash

# Run cmd
$ docker-compose up -d

# Create .env By Copy .env.example

```


## Installation

```bash
# install
$ yarn install

# prisma init
$ yarn prisma db:deploy

# seed example data
$ yarn prism db:seed

```

## Running the app

```bash
# development
$ yarn run start


```
